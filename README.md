# spring-boot-dubbo

Dubbo项目   

Spring-Boot整合dubbo。使用的是xml配置,同时也是用了注解。 
   
MySQL + mybatis 获取数据，数据池使用durid。在启动项目的时候，请确保本地的
**zookeeper已经启动**。
